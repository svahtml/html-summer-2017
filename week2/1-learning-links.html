Learning Links

There are two parts to a link

  • <a> - tag - defines the anchor tag.
  • href="file" - attribute - defines the location of the file we are linking to.

Building a link

When building a link you need to have both a file name and the text that will be clicked on.

Linking to a file in the same folder

When linking to a file in the same folder you need to only define the file name in the href.

Linking to a file in a subfolder

When linking to a file in a subfolder you need to define both the folder name and the file name.

Linking to a file in a subfolder is both fun and easy.

Linking to external sites or files

When linking to the "internet" you need to define the full path the the site or file.
