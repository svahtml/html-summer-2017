Learning Block Level Elements

There are a number of ways we can format our HTML page.

Paragraph Tag

The paragraph tag is used to define large blocks of text within an article or on a page.

Headline Tags

Headline tags are use define sections within an article. There are 6 version of the headline tag.

Break Tag

The break tag will add a line break with a block element.

Example

Dan Hahn
June 7, 2017

My Great Company
111 5th Ave
New York, NY 10003

Lists

List are a way to display a group of content in list form.

There are two types of list. Ordered and unordered

Ordered List

  1. Skittles
  2. Snickers
  3. Milky Way

Unordered List

  • Apple
  • Orange
  • Grape

Nested List

  1. Red
  2. Blue
    • Dark Blue
    • Sky Blue
  3. Green

Blockquote

Blockquote is used to define a large quote in an article.
